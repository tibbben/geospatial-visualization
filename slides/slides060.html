Geocoding
  • list of address information, usually comma separated values (csv)
  • ArcGIS geocoding service
    • Very accurate and up-to-date geocoding engine
    • Uses credits - see Understanding Credits
  • "Free" online geocoding services
    • Census.gov - somewhat accurate
    • Texas A&M - needs free account
    • geocode.io - first 1000/day free, then you pay
    • More, ... google "online geocoders"
  • in the GIS ...
    • Import as csv with lat/long coordinates as columns
    • Google "create x,y points from csv" in [GIS of choice]

Labeling 1
  • Besides getting data, the most difficult task in cartography
  • Label the LEEDs layer
    • QGIS: right click layer > layer properties > labels
    • ArcGIS Pro: right click layer > label
  • Select field to make label from
    • QGIS: in label properties look for fields menu
    • ArcGIS Pro: right click layer > Labeling properties
    • Note: you can make compound expressions from two or more fields
  • Set font characteristics
    • San serif for human geography, serif for natural features
    • Set positions of labels
    • Note many options: priority, stacking, and so on
Labeling 2
  • Besides getting data, the most difficult task in cartography
  • Get some data and add to your map:
    • Miami-Dade Data Portal: streets
    • Florida Fish and Wildlife Conservation Commission: Florida Shoreline
  • Explore the streets attribute table
    • Note: ST_NAME and ST_TYPE
    • Note: CLASS
    • anything else useful for labels?
  • Label the streets layer
    • choose field or create compound string
    • filter layer on CLASS=1
    • Note: very messy, to many segments
Labeling 3
  • Besides getting data, the most difficult task in cartography
  • Create a layout (review)
    • Aggregate streets layer based on ST_Name and ST_TYPE
    • QGIS: Menu 'vector' > geoprocessing -> dissolve
    • ArcGIS Pro: in toolbox search for 'dissolve'
    • ArcGIS Pro, make sure to include CLASS as statistics field
  • Create a layout (review)
    • Add locator map
    • QGIS: overviews in layout properties
    • ArcGIS Pro: add second map and use extent indicators
    • Add title, scale, legend, and north arrow
    • Add individual labels as text elements
Creating vector data 1

  • Remember: the project file/folder is NOT the data

      create new Feature Class / Shapefile
    • set the projection
    • Define the columns you want ...
Creating vector data 2
  • Make Layer editable
    • QGIS - the little yellow pencil
    • ArcGIS - Edit tab -> Create
    • QGIS - look for the editing icons
  • Add features
    • Try to maintain one scale
    • right click to finish sketch/part
    • Remember to save edits often
      layer, not the project
  • Turn off layer editing, save edits
Editing vector data
  • Remember: the project file/folder is NOT the data
  • Make Layer editable
    • QGIS - the little yellow pencil
    • ArcGIS - right click > make layer editable
  • Modify features
    • add, delete, or move entire feature
    • for one feature: add, delete, or move vertices
    • modify attribute data for individual features (in attribute table)
    • Remember to save edits often (not the project)
  • Turn off layer editing
