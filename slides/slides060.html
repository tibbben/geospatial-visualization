Georeferencing #2 ARC GIS Pro
  • For both raster (tiff, jpg) and vector (dxf) data
  • ArcGIS Pro Raster
    • add layer to map
      If there is a question about statistics read it carefully, explore the options, look it up on google, make an informed decision ...
    • click Raster layer in table of contents: find "Imagery" panel > georeference
    • fit to display, move, scale
    • in layer symbology find the "Mask" panel and display the background 255, 255, 255 as transparent
    • add control points - you need at least three
      Note that there are many "transformations" available (see the menu). If you are not sure, consult google.
    • open the "control point table" to better understand the process
    • save the layer (remember layer not project)
Georeferencing #3 ARC GIS Pro
  • Publish the layer to ArcGIS Online
    • Remove all base map layers from the table of contents
    • Find the Share tab, make sure you are logged in to your account (upper right)
    • Share as a "Web Layer" - "Publish Web Layer"
    • Name the layer: "UM2022Drone_your_initials", give it a short summary and a tag or two
    • Make it a "Tile" layer, save it to a new folder called "myCollector", and share with everyone
    • in the "Configuration" tab set the visibility from level 14 to 21 (IMPORTANT, do not go higher than 21)
    • click "Analyze" and then "Publish"
Georeferencing #4 ARC GIS Pro

    • load image in QGIS
ArcGIS Field App 1
  • Similar to create polygon layer (back two slides)
    > *but* do this in ArcGIS Online
  • Login and go to main "Contents" tab:
    • Navigate to your "myCollector" folder
    • + New Item > Create Feature Layer
    • Define your own layer > call it "um_trees_your_initials" and make it a "point Layer" > NEXT
    • Add tags, summary, and click SAVE
  • Finish setting up your layer
    • on the DATA tab add fields (see next slide)
    • on the SETTINGS tab approve "Public Data Collection" > SAVE
    • on the OVERVIEW tab click the "Share" button, make public "Everyone (public)"
  • Turn off layer editing, save edits
ArcGIS Field App 2
  • Create several columns/fields for:
    • Family
    • Species
    • Common name
    • Height
    • Value
    • Canopy Area
  • Pay attention to data types:
    • text
    • integer
    • double
    • numeric
    • float
    • blob
  • Turn off layer editing, save edits
Edit the field "Family" to have a controlled vocabulary:
  • on the DATA tab edit FIELDS and choose "Family"
  • click the "Create List" button and enter the following:
    • Meliaceae
    • Burseraceae
    • Fagaceae
ArcGIS Field App 3
  • Create a new ArcGIS Online Map
  • Go to main "Map" tab and click "Open in Map Viewer Classic":
    • rename your map (upper left corner)
    • add your new tree layer with the add button

    • save the map as "TreeCollector_YourInitials"
    • navigate to the map in the content pane and share the map with "everyone"
  • Turn off layer editing, save edits
